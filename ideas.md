# 心理语言对战RPG - 设计思路

## 1. 赛博故障 (Cyber Glitch)
<response>
<text>
**Design Movement**: Glitch Art / Cyberpunk
**Core Principles**:
1.  **不稳定美学**: 利用视觉上的错位、噪点和色彩分离来传达心理的不稳定和冲突。
2.  **高对比度**: 深色背景与高饱和度的霓虹色（酸性绿、电光紫、故障红）形成强烈对比。
3.  **原始数据感**: 字体和UI元素模仿终端代码、原始数据流，强调“语言即代码，攻击即黑客”的概念。
4.  **沉浸式压迫**: 通过背景的暗色调和Boss的视觉中心地位，营造一种被审视、被压迫的氛围。

**Color Philosophy**:
*   **Background**: 深邃的黑紫色 (#0a0a12) 或深蓝黑色，模拟深夜或潜意识深处。
*   **Primary**: 电光紫 (#b026ff) 和 酸性绿 (#39ff14)，代表精神污染和心理攻击。
*   **Accent**: 故障红 (#ff0055) 用于暴击和伤害，青色 (#00ffff) 用于理性和逻辑。
*   **Text**: 亮白色或带有轻微色差的灰色，确保在深色背景下的可读性。

**Layout Paradigm**:
*   **非对称平衡**: Boss位于上方中心，但Statement Bubbles（言弹）随机漂浮或以不稳定轨迹运动。
*   **卡牌区**: 下方手牌区域采用类似HUD（平视显示器）的布局，带有科技感的边框和扫描线效果。
*   **破碎感**: UI组件边缘可能带有锯齿或断裂效果，模拟心理防线的崩溃。

**Signature Elements**:
1.  **CRT扫描线**: 覆盖在整个画面或特定UI元素上的细微横纹。
2.  **色差偏移 (Chromatic Aberration)**: 关键文本或攻击特效带有RGB分离效果。
3.  **数据碎片**: 背景中漂浮的随机字符或代码片段。

**Interaction Philosophy**:
*   **点击反馈**: 点击卡牌时产生“电流”或“数据传输”的视觉效果。
*   **拖拽**: 拖拽卡牌时，连接线呈现为不稳定的光束。
*   **击碎**: Statement被击破时，像玻璃一样碎裂，伴随故障闪烁。

**Animation**:
*   **Boss呼吸**: Boss形象（背景图）带有轻微的缩放或扭曲，模拟呼吸或不稳定的存在感。
*   **文字显现**: 文字以打字机效果出现，偶尔出现乱码然后修正。
*   **卡牌悬停**: 卡牌悬停时产生高亮和轻微的抖动。

**Typography System**:
*   **Display**: 选用带有科技感或像素感的字体（如 "Press Start 2P" 或类似风格的无衬线字体），用于标题和暴击数字。
*   **Body**: 清晰的无衬线字体（如 "Roboto Mono" 或 "JetBrains Mono"），保证长文本的可读性，同时保持代码感。
</text>

## 2. 潜意识深渊 (Subconscious Abyss)
<response>
<text>
**Design Movement**: Surrealism / Dark Fantasy
**Core Principles**:
1.  **梦境逻辑**: 视觉元素漂浮、模糊，缺乏明确的物理边界。
2.  **有机纹理**: 使用烟雾、墨迹或水流的纹理，而非硬朗的几何线条。
3.  **情感映射**: 颜色随Boss情绪（阶段）变化，从压抑的灰蓝到愤怒的深红。
4.  **极简主义**: 减少UI干扰，让文字和背景图成为绝对焦点。

**Color Philosophy**:
*   **Background**: 极深的墨蓝色或黑色，带有流动的烟雾纹理。
*   **Primary**: 苍白的金色或银色，代表微弱的理智之光。
*   **Accent**: 血红色用于伤害，深紫色用于恐惧/威胁。
*   **Text**: 柔和的乳白色，带有发光效果。

**Layout Paradigm**:
*   **中心辐射**: Boss位于绝对中心，Statement Bubbles 像鬼火一样围绕Boss旋转。
*   **悬浮卡牌**: 手牌像塔罗牌一样悬浮在下方，边缘模糊发光。

**Signature Elements**:
1.  **烟雾缭绕**: 动态的烟雾遮罩。
2.  **墨迹晕染**: UI背景采用半透明的墨迹效果。
3.  **光晕**: 关键元素周围有柔和的光晕。

**Interaction Philosophy**:
*   **流动**: 卡牌拖拽时像是在水中移动，有阻力和波纹。
*   **消散**: Statement被击破时化为烟雾消散。

**Animation**:
*   **漂浮**: 所有元素都在缓慢漂浮。
*   **淡入淡出**: 界面切换柔和缓慢。

**Typography System**:
*   **Display**: 衬线字体（如 "Cinzel" 或 "Playfair Display"），营造古老、神秘感。
*   **Body**: 优雅的衬线或无衬线字体，注重阅读体验。
</text>

## 3. 霓虹黑色电影 (Neon Noir)
<response>
<text>
**Design Movement**: Film Noir / Synthwave
**Core Principles**:
1.  **光影叙事**: 强烈的明暗对比（Chiaroscuro），利用阴影隐藏信息，利用光线强调重点。
2.  **都市孤独**: 营造一种雨夜、霓虹灯、孤独侦探的氛围。
3.  **复古未来**: 结合80年代的复古审美和现代UI交互。
4.  **硬派侦探**: UI元素模仿老式档案、录音带或证据板。

**Color Philosophy**:
*   **Background**: 深蓝灰色，带有雨丝效果。
*   **Primary**: 霓虹粉和霓虹蓝。
*   **Accent**: 警示黄。
*   **Text**: 打字机风格的灰白色。

**Layout Paradigm**:
*   **档案式**: 屏幕看起来像是一个正在整理案件的桌面。
*   **卡牌**: 设计成拍立得照片或证据卡片。

**Signature Elements**:
1.  **百叶窗阴影**: 画面上覆盖百叶窗投下的阴影。
2.  **雨滴**: 屏幕前景有雨滴滑落的效果。
3.  **霓虹边框**: 关键区域用霓虹灯管风格的边框。

**Interaction Philosophy**:
*   **机械感**: 点击按钮有机械开关的声音和视觉反馈。
*   **盖章**: 攻击判定像是在文件上盖章。

**Animation**:
*   **闪烁**: 霓虹灯管的闪烁效果。
*   **滑动**: 元素像纸张一样滑动进入。

**Typography System**:
*   **Display**: 粗犷的无衬线字体或霓虹灯管字体。
*   **Body**: 打字机字体（如 "Courier Prime"）。
</text>
